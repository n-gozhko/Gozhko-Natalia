Начнем с самых азов программирования.

Нас интересует одна плата из компьютера - это оперативная память, потому что именно она взаимодействует с браузером.
Например, нам нужно посчитать, сколько бдет 2+2, то в оперативной памяти эти значения нужно сохранить

Есть два вида устройств памяти

  • Постоянное запоминающее устройство - винчестер, жесткий диск, ssd - здесь можно что-то записать и долгое время ти записи хранить
  • Оперативная память - она работает только до момента перезагрузки. Обновили - память обнулилась

Эта оперативная память состоит из таких сот - ячеек, они называются кластерами и информация там представлена в виде 0 и 1

В JS мы будем какие-то куски памяти объединять, например большая строчка А и потом к ней как-то обращаться

Переменные

Переменная — участок памяти компьютера (браузера), который имеет имя и содержит некоторое значение. Значение переменной может изменяться.

Перед тем как использовать переменную, её нужно объявить:
var имяПеременной;

Объявленная таким образом переменная не содержит значения.

Можно объявить переменную, сразу присвоив ей значение:
var имяПеременной=значение;

Можно присвоить новое значение ранее объявленной переменной:
имяПеременной=значение;

Логические операции (или по битому или бинарные операции)

&& - это значение равно И

Т.е. мы можем задать две переменные, в консоле их прописать через И, одно из выражением будет true, а второе будет false.
По итогу, консоль нам покажет false, т.к. и должны быть верные оба вырежния, т.к. у нас задан И

Также можно задать ИЛИ - через ||. Тут уже одно из выражение должно быть true - и тога консоль покажет true

Также есть еще !, т.е. мы инвертируем наше булевое значение - логическое. Т.е. если у нас значение на самом деле было true, то черезе ! оно инвертируется и консоль выдаст false

Оператор if (условный оператор)

Условный оператор — оператор, который позволяет выполнить (или не выполнить) некоторые действия (операторы) в зависимости от истинности условия.

Он говорит о том, что у нас есть какое-то условие, если оно выполняется, то, что находится в операторе

Пример ниже - если наше число будет больше 18, то консоль пропишет я большой.

Если у нас оператор один, то его можно записать в строчку в if, если несколько, то только в скобках

В последнем случае, вторая консоль не относится к if

Синтаксис:

if ( условие )
оператор

если условие истинно, оператор будет выполнен, если ложно — не будет выполнен.
Операторы до и после if (т.е. не показанные в образце) будут выполнены в любом случае.

В данно случае мы говорим, что если условие соблюдено - отрабатывает первый консоль, если не соблюдено - то else

Вторая форма синтаксиса:

if ( условие )
оператор1
else
оператор2

если условие истинно, будет выполнен оператор1, если ложно — будет выполнен оператор2.
Операторы до и после if (т.е. не показанные в образце) будут выполнены в любом случае.

Если необходимо при истинности или ложности условия выполнить не один оператор, а несколько, можно применять составной оператор.
Составной оператор — это любая группа операторов, взятая в фигурные скобки.

if ( условие ) {
оператор1
оператор2
...
else {
оператор3
оператор4
...
}

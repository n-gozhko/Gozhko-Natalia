Тег picture

Представляет собой контейнер для хранения нескольких или ноль элементов source, которые поддерживают элемент img и прописываются перед ним. Это позволяет указывать разные изображения с учётом размера экрана, плотности пикселей, формата изображения и других параметров. Браузер рассмотрит каждый из дочерних элементов source и выберет один, соответствующий лучшему совпадению; если совпадений среди элементов source найдено не будет, то будет выбран файл, указанный атрибутом src элемента img. Затем выбранное изображение отображается в пространстве, занятом элементом img. Нет совместимости с Internet Explorer

Синтаксис:

    picture
        source
        img
    /picture

Атрибуты тега source:

  • Атрибут srcset - список из одной или нескольких строк, разделённых запятыми, определяющий набор возможных изображений, представленных для отображения в браузере.
  • Атрибут media - позволяет определить медиавыражение (высота, ширина и т.д.), которое веб-браузер будет анализировать для выбора элемента source.
  • Атрибут type - позволяет вам обозначить MIME-тип данных, указанных в атрибуте srcset элемента source. Если браузер не поддерживает данный тип, то элемент source опускается.
  • Атрибут src - указывает URL адрес медиа-файла.
  • Атрибут sizes задаёт пространство, которое изображение будет занимать на экране. Например sizes="(min-width: 1280px) 1200px," - изображение займёт до 1200px, если минимальная ширина экрана равна 1280px.

Стилевое свойство list-style-type

Это вид маркера списка.
        Может применяться к элементам ul и ol, а также к li.
        Некоторые возможные значения:
        none — нет маркера
        disc — чёрный кружок
        circle — светлый кружок
        square — чёрный квадратик
        decimal — десятичные числа (1, 2, 3, ...)
        decimal-leading-zero — десятичные числа с ведущим нулем (01, 02, 03, ... 98, 99)
        lower-roman — строчные римские числа (i, ii, iii, iv, v, ...)
        upper-roman — прописные римские числа (I, II, III, IV, V, ...)
        lower-greek — строчные греческие буквы
        lower-latin — строчные латинские буквы (a, b, c, ..., z)
        upper-latin — прописные латинские буквы (A, B, C, ..., Z)

Стилевое свойство cursor — форма курсора мыши. Задаёт форму, которую принимает курсор мыши при поднесении к данному элементу.

  • форма курсора определяется автоматически
  • курсор операционной системы по умолчанию; в Windows — стрелка
  • для данного элемента можно получить справку. стрелка с вопросом
  • элемент можно перемещать либо элемент уже перемещается
  • на элемент можно нажимать/щёлкать. рука

Стилевые свойства для управления шрифтом

Стилевое свойство line-height — высота строки (интервал между строками). Задаёт минимальную высоту строк для шрифта, которым будет отображаться содержимое элемента.

  • line-height: размер — явно задаёт высоту строки.
  • line-height: число — задаёт высоту строки, равную размеру шрифта, умноженному на указанное число.
  • line-height: normal — обычная высота строки, зависящая от размера шрифта.

Стилевое свойство font-weight — жирность шрифта. Задаёт жирность шрифта, которым будет отображаться содержимое элемента.

  • font-weight: bold — жирный шрифт.
  • font-weight: normal — нормальный (нежирный) шрифт.
  • font-weight: bolder — жирнее чем у контейнера. Также допустимо значение lighter — светлее (менее жирно) чем у контейнера. Не рекомендуется использовать.
  • font-weight: число — явное задание жирности. Могут быть указаны значения 100, 200, 300, 400, 500, 600, 700, 800, 900. Их трактовка зависит от браузера, поэтому их не рекомендуется использовать.

Стилевое свойство font-style — стиль шрифта.

  • font-style: italic — курсивный шрифт. Его начертание, в общем случае, отличается от обычного, если используемый шрифт содержит символы такого начертания.
  • font-style: oblique — наклонный шрифт. Его начертание совпадает с обычным, символы обычного начертания просто наклоняются вправо. Если используемый шрифт содержит не оба начертания (italic и oblique), а только одно из них (а таково большинство шрифтов), браузер будет использовать имеющееся начертание, независимо от того, какое значение установлено стилевому свойству font-style — italic либо oblique.
  • font-style: normal — обычный шрифт.

Стилевое свойство font-variant — устанавливает способ отображения строчных букв.

  • font-variant: normal — строчные буквы отображаются обычным способом.
  • font-variant: small-caps — ВМЕСТО строчных букв отображаются заглавные, уменьшенные до размера строчных.

Стилевое свойство letter-spacing — интервал между буквами.

  • letter-spacing: размер — задаёт интервал, который может быть как положительным, так и отрицательным.
  • letter-spacing: normal — обычный интервал.

Стилевое свойство word-spacing — интервал между словами.

  • word-spacing: размер — задаёт интервал, который может быть как положительным, так и отрицательным.
  • word-spacing: normal — обычный интервал.

Шрифты имеют родовое имя. Это семейтсов шрифта. Туда относят все максимально похожие между собой шрифты. И если наш шрифт по какой-то причине не заработает - мы можем взять другой шрифт из этого семейства. Обычно прописывается имя шрифта, затем его семйство

Стилевое свойство font позволяет задать несколько реквизитов шрифта сразу, одной записью. Размер шрифта и имя шрифта должны быть заданы обязательно. Синтаксис: font: [font-style] [font-variant] [font-weight] font-size [/line-height] font-family

SVG — пример

SVG (от англ. Scalable Vector Graphics — масштабируемая векторная графика) — язык разметки масштабируемой векторной графики.
Предназначен для описания двумерной векторной или векторно/растровой графики в формате XML (а то есть на языках, родственных языкам HTML и CSS). Поддерживает как неподвижную, так и анимированную интерактивную графику, т.е. элементами SVG-графики можно управлять средствами CSS или JavaScript.

stroke - тоже, что и бордер, задает цвет границ.
stroke-width - задает толщину границ графики.
cx и cy - расположение графики по x и y (Чем больше по y - тем ниже расположено. Чем больше по x - тем правее расположено)
fill-opacity - задает прозрачность элементов

Поддержка: Chrome 27+, Firefox 29+, IE 9+, Safari 5.1+, iOS Safari 6.1+, Android Browser 4+
